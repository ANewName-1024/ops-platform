# OPS 平台前端设计方案

## 一、系统架构

### 1.1 技术栈
- **框架**: Vue 3 + Vite
- **UI**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **图表**: ECharts
- **图标**: Element Plus Icons

### 1.2 目录结构
```
frontend/src/
├── api/                    # API 接口
│   ├── auth.js            # 认证
│   ├── knowledge.js       # 知识库
│   ├── workflow.js         # 工作流
│   ├── autoheal.js        # 自愈
│   ├── grayrelease.js     # 灰度发布
│   ├── dashboard.js       # 监控
│   ├── ai.js              # AI
│   ├── notification.js    # 通知
│   ├── certificate.js     # 证书
│   ├── ops.js             # 运维
│   └── rbac.js           # 权限
├── components/            # 公共组件
├── views/                 # 页面
│   ├── layout/           # 布局
│   ├── login/            # 登录
│   ├── dashboard/        # 首页
│   ├── knowledge/        # 知识库
│   ├── workflow/         # 工作流
│   ├── autoheal/         # 自愈
│   ├── grayrelease/      # 灰度发布
│   ├── monitor/          # 监控中心
│   ├── ai/               # AI 助手
│   ├── notification/     # 通知中心
│   ├── certificate/      # 证书管理
│   ├── ops/              # 运维中心
│   └── admin/            # 系统管理
└── styles/               # 样式
```

## 二、色彩与平面设计

### 2.1 色彩系统

#### 主色调
| 用途 | 颜色 | HEX |
|------|------|-----|
| 主色 | 渐变紫蓝 | #667eea → #764ba2 |
| 主色浅 | 浅紫 | #8B5CF6 |
| 主色深 | 深紫 | #5B21B6 |

#### 功能色
| 用途 | 颜色 | HEX |
|------|------|-----|
| 成功 | 绿色 | #10B981 |
| 警告 | 橙色 | #F59E0B |
| 危险 | 红色 | #EF4444 |
| 信息 | 蓝色 | #3B82F6 |

#### 中性色
| 用途 | 颜色 | HEX |
|------|------|-----|
| 文字主色 | 深灰 | #1F2937 |
| 文字次要 | 灰色 | #6B7280 |

- 主字体: Inter, -apple-system, BlinkMacSystemFont, sans-serif

- 基础单位: 4px
- 常用间距: 4, 8, 12, 16, 24, 32, 48px

## 三、响应式布局设计

### 3.1 断点
| 设备 | 宽度 | 布局 |
|------|------|------|
| 手机 | < 768px | 单列，底部导航 |
| 平板 | 768-1024px | 双列，侧边栏折叠 |
| 桌面 | 1024-1440px | 标准三栏 |
| 大屏 | > 1440px | 宽屏三栏 |

## 五、交互设计

### 5.1 通用交互规范

#### 按钮
- 主按钮: 背景渐变 #667eea → #764ba2，圆角 8px
- 次按钮: 边框 #E5E7EB，背景白色
- 危险按钮: 背景 #EF4444

#### 表格
- 斑马纹: 奇数行 #F9FAFB
- 悬停: 行背景 #F3F4F6
- 选中: 边框左侧 3px #667eea

#### 表单
- 输入框: 圆角 8px，边框 #E5E7EB
- 焦点: 边框 #667eea，阴影 0 0 0 3px rgba(102,126,234,0.1)

### 5.2 动效设计
- 页面切换: 淡入淡出 0.3s
- 按钮悬停: transform scale(1.02)，0.2s
- 卡片悬停: 阴影增强，0.3s
- 加载: 骨架屏 + 脉冲动画
