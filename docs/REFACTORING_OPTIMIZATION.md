# 重构方案优化建议

**审核日期：2026-03-19**
**审核人：小爪**

---

## 1. 服务拆分优化

### 当前方案
```
workflow-service (8081)
knowledge-service (8082)  
chat-service (8083)
```

### 优化建议

| 问题 | 建议 | 理由 |
|------|------|------|
| knowledge + AI 耦合 | 拆分为 knowledge-service + ai-service | AI 能力可复用 |
| notification 缺失 | 添加 notification-service | 通知独立部署 |
| admin 过于简单 | 并入 gateway 或独立 | 根据实际复杂度 |

### 优化后

```
gateway (8080)              # 网关 + admin
workflow-service (8081)    # 工作流
knowledge-service (8082)  # 知识库存储
ai-service (8083)         # AI 能力 (可复用)
chat-service (8084)       # 对话
notification-service (8085) # 通知
```

## 2. 数据层优化

### 当前问题
- 单一数据库
- 无读写分离
- 无分库分表

### 优化建议

| 方案 | 描述 | 复杂度 |
|------|------|--------|
| 读写分离 | 主从 PostgreSQL | 中 |
| 多租户 | 按 tenant_id 分表 | 高 |
| 分库 | 按业务拆分 | 中 |

### 推荐：渐进式

```
Phase 1: 单一数据库 + 分表
Phase 2: 读写分离
Phase 3: 按服务分库
```

## 3. 安全设计增强

### 当前缺失

| 缺失项 | 建议 |
|--------|------|
| API 签名验签 | 添加请求签名机制 |
| 敏感数据加密 | 字段级加密 |
| 审计日志 | 完整操作日志 |
| 限流熔断 | Spring Cloud CircuitBreaker |
| 黑名单/IP 白名单 | 基础防护 |

### 优化架构

```
Client -> WAF -> Gateway -> Auth -> Service
                       |
                 Rate Limiter
                 Circuit Breaker
                 Audit Log
```

## 9. Nacos 集成详情

### Nacos 核心功能

| 功能 | 描述 |
|------|------|
| **服务发现** | 自动注册/注销、负载均衡、心跳检测 |
| **配置管理** | 动态配置、版本管理、配置监听 |
| **命名服务** | DNS 解析、流量管理 |

### 配置示例

```yaml
# bootstrap.yml
spring:
  cloud:
    nacos:
      server-addr: nacos.openclaw.ai:8848
      username: ${NACOS_USERNAME:nacos}
      password: ${NACOS_PASSWORD:nacos}
      discovery:
        namespace: prod
        group: OPENCLAW_AI_GROUP
      config:
        namespace: prod
        group: OPENCLAW_AI_GROUP
        file-extension: yml
        refresh-enabled: true
```

### 配置分组

| 服务 | Data ID | Group |
|------|---------|-------|
| gateway | gateway.yml | OPENCLAW_AI |
| workflow | workflow.yml | OPENCLAW_AI |
| knowledge | knowledge.yml | OPENCLAW_AI |
| ai | ai.yml | OPENCLAW_AI |
| chat | chat.yml | OPENCLAW_AI |
| notification | notification.yml | OPENCLAW_AI |
