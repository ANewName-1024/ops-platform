## 微服务列表

| 服务名 | 端口 | 职责 | 技术栈 |
|--------|------|------|--------|
| ops-service | 8083 | 核心运维功能 | Spring Boot + JPA |
| workflow-service | 8091 | 工作流引擎 | Spring Boot |
| knowledge-service | 8092 | 知识库管理 | Spring Boot |
| notification-service | 8094 | 消息通知 | Spring Boot |
| chat-service | 8093 | AI 对话 | Spring Boot + LLM |

## 数据流

```
用户操作 → 前端(Vue) → 请求封装(request.js) 
    → Nginx(:80) → 后端服务(8083/8091/8092/8094)
    → Controller → Service → Repository 
    → PostgreSQL/Redis → Response → 前端 → 页面更新
```
